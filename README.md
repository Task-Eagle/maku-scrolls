# maku-scrolls
maku-scrolls.html. Open in Browser: Double-click the file to open it in Chrome, Firefox, or Edge. Write: Start typing your stories about the Lion King or the wolves! Save: Click the "Save .docx" button in the top right to download your document. 
The "Ribbon" Header: Instead of a floating toolbar, I created a top bar that mimics the Word ribbon with tabs (Home, Insert, Layout) and a clean blue accent color (#2b579a).
The "Paper" Canvas: I adjusted the shadow and margins to make the editor look like a physical sheet of paper sitting on a gray desktop.
Typography: Enforced the Calibri font stack strictly, which is the default for modern Word documents.
Maku's Touch: Added a subtle header inside the editor that says "Maku's Scrolls" to personalize it for your story.
Refined Save Logic: Ensured the exported .docx carries the correct styling.
